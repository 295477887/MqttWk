# MqttWk

* t-io 版本 3.1.2.v20180712-RELEASE [https://gitee.com/tywo45/t-io](https://gitee.com/tywo45/t-io)
* kafka 版本 2.12-1.1.0 [http://kafka.apache.org/downloads](http://kafka.apache.org/downloads)
